# 🔎 Python Port Scanner

A multi-threaded TCP port scanner built with Python using socket programming.

## 🚀 Features
- TCP port scanning
- Multi-threaded scanning (fast)
- Scan single host and port ranges
- Error handling and logging
- Result export to log file

## 🧠 Technologies Used
- Python 3
- socket
- concurrent.futures
- threading
